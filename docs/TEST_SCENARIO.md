# MICEstrator 통합 테스트 시나리오

## 시나리오 개요

**행사명:** 2026 대한민국 AI 리더십 포럼 (Korea AI Leadership Forum 2026)  
**주제:** 생성형 AI 실무 도입과 조직 혁신  
**규모:** 중규모 (200~300명), 서울 코엑스 그랜드볼룸  
**목적:** 국내 중견·대기업 임원 및 AI 실무 리더를 위한 B2B 네트워킹 + 인사이트 컨퍼런스

---

## 사전 준비

### 1. 이벤트 생성

**URL:** `http://localhost:3000`

메인 화면에서 **"+ 새 행사 만들기"** 클릭 후 입력:

| 필드 | 입력값 |
|------|--------|
| 행사명 | `2026 대한민국 AI 리더십 포럼` |

→ **"시작하기"** 클릭  
→ Phase 1 페이지로 자동 이동 (`/event/{id}/phase-1`)

## Phase 1 — 행사 기획 마스터

**URL:** `/event/{id}/phase-1`

### 입력값

| 필드 | 입력값 |
|------|--------|
| 산업/분야 | `인공지능 / 엔터프라이즈 테크` |
| 준비 기간 | `6개월` (라디오 버튼) |
| 행사 규모 | `중규모 (100~500명)` (카드 선택) |

→ **"Phase 1 실행"** 클릭 (약 15~25초 소요)

### 기대 결과

| 항목 | 예상 출력 예시 |
|------|--------------|
| 행사명 (국문) | `2026 대한민국 AI 리더십 포럼` |
| 행사명 (영문) | `Korea AI Leadership Forum 2026` |
| 슬로건 | `AI, 이제는 실행이다` |
| 핵심 키워드 | `생성형 AI`, `조직혁신`, `ROI`, `거버넌스`, `실무전략` |
| PEST 분석 | 정치·경제·사회·기술 4개 관점 분석 제공 |
| 타깃 페르소나 | 2~3개 페르소나 (예: AI 도입 담당 임원, 현업 개발팀장) |
| 기획 근거 | AI 도입 ROI 불확실성, 조직 저항 등 Pain Point 기반 설명 |

### 확인 포인트
- [ ] 행사명 카드가 표시되는가
- [ ] 슬로건과 부제가 생성되었는가
- [ ] PEST 그리드 4개 항목이 모두 채워졌는가
- [ ] 페르소나 아코디언이 펼쳐지는가

| 비즈니스 ROI | `투자 대비 약 4.2배의 비즈니스 파이프라인 확보` |

## 전체 파이프라인 흐름 요약

```
[메인 화면]
   │
   ▼ 행사 생성 → event_id 발급
   │
[Phase 1] ──▶ event_master 생성 (DB: phase_results #1)
   │              ↓ 자동 주입
[Phase 2] ──▶ WBS + 마일스톤 (DB: phase_results #2)
   │
[Phase 3] ──▶ 브랜드 아이덴티티 (DB: phase_results #3 + brand_memory)
   │                                          ↓ 자동 주입
[Phase 4] ──▶ 연사 섭외 이메일 (DB: phase_results #4)
   │
[Phase 5] ──▶ 마케팅 에셋 (DB: phase_results #5)
   │
   │ [행사 개최]
   │
[Phase 6] ──▶ ROI 분석 보고서 (DB: phase_results #6)
```

## 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| `Phase 1 결과가 없습니다` 오류 | Phase 1을 실행하지 않음 | Phase 1 먼저 실행 |
| AI 응답이 스키마를 만족하지 않습니다 | GPT-4o 출력 불안정 (드물게 발생) | **재생성** 버튼 클릭 |
